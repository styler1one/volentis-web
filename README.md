# Volentis.ai Website

EU-sovereign enterprise AI platform website.

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript (strict)
- Tailwind CSS
- next-intl for i18n (EN, NL, DE, FR)
- Vercel for deployment

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
